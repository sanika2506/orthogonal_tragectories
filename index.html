How it works: For a curve family F(x,y,C)=0, orthogonal trajectories are curves that intersect the family at right angles. They satisfy dy/dx = -1/(dy/dx of original).

Calculation Details

Selected Curve: Circle Family

Parameter Value: C = 3

Step 1: Original Curve Family

The family of curves is given by: x² + y² = C

This represents concentric circles centered at the origin with radius √C.

Step 2: Derivative Calculation

Using implicit differentiation: 2x + 2y(dy/dx) = 0

Solving for dy/dx: dy/dx = -x/y

Step 3: Orthogonal Condition

For orthogonal trajectories, slope₁ × slope₂ = -1

If original slope = -x/y, then orthogonal slope = y/x

Step 4: Solution

Solving dy/dx = y/x gives: y = mx

This represents straight lines through the origin (radial lines).
